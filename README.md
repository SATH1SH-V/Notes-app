# Notes App

A simple and responsive **Notes App** built with **HTML, CSS, JavaScript, Express.js, and MongoDB (Mongoose)**. The app allows users to create, view, edit, and delete notes with data stored in a database for persistence.

## Features
- Create, edit, and delete notes.
- Responsive and clean frontend design.
- **Express.js** backend for handling API requests.
- **MongoDB** database integration via **Mongoose**.
- Persistent storage of notes across sessions.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)
